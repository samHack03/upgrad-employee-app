# Find Your Bank

Find your bank app

### Technology Used



- [React](https://reactjs.org/)
- [React Router](https://github.com/reactjs/react-router)
- [Axios](https://github.com/axios/axios)
- [Material UI](https://mui.com/)


### To Run the project




- Clone the repo

```shell
https://github.com/Sg1211/Find-Your-Bank-Groww.git
cd folder-name
```





- Installation

```shell
npm install
```




- Running Dev Server

```shell
npm start
```
